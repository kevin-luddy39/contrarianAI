Bell Tuning Cloud

Managed monitoring for AI context-window health. Aggregates outputs from context-inspector, retrieval-auditor, tool-call-grader, and predictor-corrector.

Get started

No workspaces yet. Create one with the admin API:

curl -X POST http://localhost:4200/api/workspaces \
  -H 'x-admin-token: dev-admin-token' \
  -H 'content-type: application/json' \
  -d '{"name":"acme-corp"}'

Or seed the demo data:

npm run seed

Then refresh this page.
